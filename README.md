LabelFlow is a full-featured record label management platform built for modern independent labels. Manage artists, plan campaigns, track analytics, generate AI content, scout street marketing locations, and run complete label operations — all in one sleek React 19 app.

## ✨ 15 Modules

| Module | Description |
|---|---|
| **Dashboard** | Real-time stats, activity feed, upcoming events, AI insights |
| **Artists** | Roster cards with streaming stats, growth trends, artist profiles |
| **Campaigns** | Track marketing campaigns — progress, budgets, performance |
| **Assets** | Filter and organize audio, images, video, documents |
| **Schedule** | Interactive calendar with drag navigation |
| **Contacts** | Industry CRM — managers, promoters, radio, press, brands |
| **Opportunities** | AI-powered sync deals, festivals, grants, playlist placements |
| **Notes** | Categorized notes for meetings, ideas, campaigns |
| **To-Do & Workflows** | Drag-and-drop Kanban board with priority-color task cards |
| **AI Artist Development** | Generate artist profile, album concept, and merch line with AI |
| **AI Blog Writer** | Rich-text editor with AI generation, chat assistant, blog library |
| **AI Merch Mockups** | Generate product mockups — tees, hoodies, mugs, posters, more |
| **Resources** | Categorized link/document library |
| **Street Marketing** | AI-powered city location scouting for poster drops |
| **Analytics** | Chart.js streaming growth, audience geography, revenue mix |

## 🚀 Quick Start

```bash
git clone https://github.com/RhythrosaLabs/labelflow.git
cd labelflow
npm install
npm run dev
```

Or try the [live demo](https://rhythrosalabs.github.io/labelflow).

## 🛠️ Tech Stack

- **React 19 + TypeScript** — component architecture
- **Vite 7** — build tool
- **Tailwind CSS** — styling
- **Chart.js** — analytics visualizations
- **OpenAI** — AI content generation
